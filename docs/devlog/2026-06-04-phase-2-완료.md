## 완성된 기능

### 자재 등록 폼 (/items/new)
- 거래 종류 토글: 구해요 / 나눔 / 판매중
- 사진 업로더 (다중 선택, 미리보기, 순서 변경, "대표" 배지, 최대 10장)
- 제목 + 자재 정보 4필드 (명칭/규격/수량/단위)
- 카테고리 다중 선택 (최대 3개)
- 가격 + 가격 협의 (자동 콤마, free일 땐 숨김)
- 상세 설명
- 지역 3택 (제주시/서귀포시/제주 전체) + 이동 가능 메모
- 운반 옵션 다중 선택
- 전화번호 (자동 하이픈)
- Zod 검증 + 인라인 에러 메시지

### 백엔드 흐름
- 사진 Supabase Storage 업로드 (브라우저 직접, RLS로 본인 폴더만)
- createItem Server Action
  - items 테이블 INSERT
  - item_categories 조인 테이블 INSERT
  - item_images 테이블 INSERT
  - 실패 시 자동 롤백 (DB 행 삭제 + Storage 파일 정리)
- 성공 시 /items/[id] 상세 페이지로 이동

### 진입점
- 플로팅 + 버튼 (전역 레이아웃, /items/new 에서만 숨김)

### 가드
- requireUser 가드: 비로그인 시 로그인 페이지로 리다이렉트

## 주요 의사결정

1. **카테고리 최대 5개 → 3개로 변경** (Phase 2 도중 결정)
   - 사용자 부담 감소 + 검색 정확도 향상

2. **지역 43개 읍면동 → 3택으로 단순화**
   - 제주시 / 서귀포시 / 제주 전체(이동 가능)
   - 추후 세분화 필요 시 가역적 (기존 43개 데이터 보존)

3. **자재 정보 4필드 추가**
   - 제목과 명칭 분리: 제목은 마케팅성, 명칭은 검색 키워드
   - 검색·필터 정확도를 위한 구조화된 데이터

4. **사진 업로드 시점: 폼 제출 시 한꺼번에**
   - 고아 파일 방지

5. **순서 변경: ↑↓ 버튼** (드래그는 추후 검토)
