## 배운 점 / 메모

- Tailwind v4 의 `@theme` 블록에서 hex 값으로 토큰 관리하는 방식 정착
- Next.js 15 Server Component + Server Actions 조합으로 로그아웃 흐름 깔끔하게 처리
- 카카오 OAuth 세션과 짜투 세션이 분리돼 있어, 짜투에서 로그아웃해도 카카오 세션은 유지됨 → 재로그인 시 동의 화면 안 뜨고 즉시 복귀 (정상 동작이지만 처음엔 헷갈렸음)
- `requireUser(redirectTo)` 가드 시그니처: Server Component에서 현재 경로를 자동으로 알 수 없어, 호출하는 쪽이 인자로 넘기는 방식 채택
